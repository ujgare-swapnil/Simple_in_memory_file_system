# Simple_in_memory_file_system
It is the simple in memory file system. study material for beginners
1) clone the Repository
2) go into my_ramfs directory.
        eg. cd my-ramfs
3) compile the code.
        eg. make all
4) install the module using following command.
        eg. make install
5) Use the file system like ramfs

6) uninstall the modules using.
        make uninstall
